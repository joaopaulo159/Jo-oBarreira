Linguagens de programação mais usadas do Brasil durante 2021

  • Javascript

    • JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma.

  • Sass CSS

    • Sass é uma linguagem de folhas de estilo concebida inicialmente por Hampton Catlin e desenvolvida por Natalie Weizenbaum. Depois de suas versões iniciais, Weizenbaum e Chris Eppstein continuaram a estender Sass com SassScript, uma simples linguagem de script usada em arquivos Sass.

  • Blade

    • O Blade é um dos template engine PHP mais usados do mercado, pois ele já vem pronto para usar no Laravel que é um dos frameworks mais baixados do mundo. Uma das vantagens do Blade é que ele não impacta no desempenho da aplicação, já que ele cria cache para arquivos PHP de suas views

  • HashiCorp Configuration Language (HCL)

    • Kit de ferramentas para a criação de linguagens de programação estruturadas.

  • Elixir

    • Elixir é uma linguagem de programação funcional, concorrente, de propósito geral que executa na máquina virtual Erlang (BEAM).Estilo de tipagem forte, dinâmica.
